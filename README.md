# 온라인 쇼핑몰 API
스프링 부트로 구현한 온라인 쇼핑몰 API입니다.

## 사용 기술
Java, Spring Boot, JPA/Hibernate

## 진행사항
1) JWT토큰을 이용하여 사용자의 인증과정을 처리하였습니다
2) Redis를 이용하여 장바구니를 구현하여 DB로부터 접근을 줄이도록 구현하였습니다.
3) Jenkins를 사용하여 CI/CD 환경을 구축하였습니다.
